# Opticalflow-framework-2D
Notebook completo per valutare optical flow 2D con RAFT, SPyNet, PWCNet e FlowNet2 su Sintel, KITTI, FlyingChairs e VKITTI2. Fornisce loader uniformi, metriche (EPE, Fl-all, Angular Error), supporto FAST/FULL, esportazione CSV/LaTeX, visualizzazioni e confronto finale dei modelli.
